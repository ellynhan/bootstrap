Basic Table

The .table class adds basic styling (light and horizontal divider) to a table

The .table-striped calss adds zebra-stripes to a table

The .table-bordered class adds borders on all sides of teh table and the cells

The .table-hover class adds a hover effect (grey background color) on table rows.

Contextual Classes

Contextual classes can be used to color the table, table rows or table cells. The classes that can be used are: .table-primary, .table-success, .table-info, .table-warning, .table-danger, .table-active, .table-secondary, .table-light and .table-dark:
